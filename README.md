# Meu Portfólio Profissional

Este é o repositório do meu portfólio pessoal, desenvolvido para centralizar meus principais projetos, habilidades e experiências na área de Tecnologia da Informação. O objetivo principal deste projeto é apresentar meu trabalho de forma clara, moderna e performática para recrutadores e profissionais do mercado.

---

## Tecnologias Utilizadas

O projeto foi construído utilizando as ferramentas mais modernas e demandadas pelo ecossistema de desenvolvimento web atual:

* **[React](https://react.dev/):** Biblioteca JavaScript focada na construção de interfaces de usuário baseadas em componentes reutilizáveis.
* **[Vite](https://vite.dev/):** Ferramenta de build ultra-rápida que otimiza o ambiente de desenvolvimento e entrega um carregamento extremamente ágil em produção.
* **[TypeScript](https://www.typescriptlang.org/):** Supersuposto do JavaScript que adiciona tipagem estática ao código, garantindo maior segurança, menos bugs e melhor manutenibilidade.
* **[Tailwind CSS](https://tailwindcss.com/):** Framework utilitário de CSS que possibilita a criação de designs customizados, responsivos e modernos diretamente nas classes do HTML/JSX.
* **[shadcn/ui](https://ui.shadcn.com/):** Coleção de componentes de interface altamente acessíveis, customizáveis e componentizados de forma nativa.

---

## Engenharia de Software Assistida por IA (AI-Assisted Development)

Alinhado com as tendências mais recentes da engenharia de software global, este projeto foi desenvolvido utilizando metodologias de **Desenvolvimento Assistido por Inteligência Artificial**. 

Durante o ciclo de desenvolvimento, ferramentas baseadas em modelos de linguagem avançados (LLMs) foram integradas ao fluxo de trabalho para atuar como um parceiro de programação (*Pair Programmer*). A IA foi utilizada de maneira estratégica para:

* **Aceleração de Prototipagem:** Geração rápida de layouts iniciais e estruturas de componentes de UI.
* **Eficiência de Código:** Refatoração de trechos lógicos, otimização de consultas e automação de tarefas repetitivas de boilerplate.
* **Resolução de Problemas:** Auxílio no diagnóstico ágil de erros e testes de compatibilidade em ambientes locais.

Essa abordagem demonstra não apenas competência técnica nas linguagens de programação, mas também a habilidade de governar e direcionar ferramentas de IA generativa para maximizar a produtividade e a entrega de valor tecnológico.
